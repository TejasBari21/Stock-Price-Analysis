# Stock Price Analysis and Prediction

## Project Overview

This project is a Python-based web application developed to analyze and predict stock market trends using historical stock data.

The application provides data visualization, stock trend analysis, and predictive insights through an interactive and user-friendly interface. It helps users understand stock market behavior and make data-driven decisions based on historical performance.

## Features

* Historical stock price analysis
* Interactive stock trend visualization
* Stock market data preprocessing and cleaning
* Price prediction using machine learning techniques
* User-friendly web interface
* Multiple stock dataset support
* Graphical representation of stock performance

## Tools Used

* Python
* Flask
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* HTML
* CSS
* JavaScript

## File Included

* `app.py` → Main Flask application
* `Datasets/` → Stock market datasets
* `templates/` → HTML templates for the web interface
* `clean.py` → Data preprocessing script

## Dashboard / Application Insights

The application provides insights such as:

* Historical stock price trends

* Data visualization through interactive charts
* Pattern identification in stock movements
* Predictive analysis based on historical data
* Comparison of stock performance metrics

## How to Run the Project

1. Install Python on your system.
2. Download or clone this repository.
3. Install the required libraries:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```text
http://127.0.0.1:5000/
```
